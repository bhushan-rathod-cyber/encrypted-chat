# 🔐 Encrypted–Decrypted Message Sender

## 📌 Overview
This project demonstrates secure communication using encryption and decryption techniques.

## 🚀 Features
- AES Encryption (Fernet)
- Secure message transmission
- Web-based chat system using Flask

## 🛠️ Technologies
- Python (Flask)
- HTML, CSS, JavaScript
- Cryptography Library

## ▶️ How to Run
1. Install dependencies:
   python -m pip install flask cryptography

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000

## 🔮 Future Scope
- End-to-End Encryption
